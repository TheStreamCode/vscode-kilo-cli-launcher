# Support

Kilo CLI launcher is maintained by Mikesoft.

## Bug Reports And Feature Requests

Use GitHub Issues for reproducible bugs, regressions, and feature requests:

- Issues: https://github.com/TheStreamCode/vscode-kilo-cli-launcher/issues

When opening an issue, include:

- VS Code version
- Operating system
- Whether you are using a single-folder or multi-root workspace
- Configured `kilocodeCliLauncher.cliCommand` value
- Steps to reproduce the problem
- Expected result
- Actual result
- What happened in the integrated terminal after launch
